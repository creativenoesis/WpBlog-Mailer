# WP Blog Mailer (Free Version)

This is the free version of WP Blog Mailer, ready for WordPress.org submission.

## Features

- Automated newsletter sending
- Basic subscriber management
- Email subscription forms via shortcode
- Basic analytics
- Cron job monitoring
- Send log tracking

## Installation

1. Upload the plugin files to `/wp-content/plugins/wp-blog-mailer/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Blog Mailer â†’ Settings to configure
4. Use shortcode `[wpbm_subscribe_form]` to add subscription forms
